# 小说写作助手 (Novel Writing Assistant)

## 简介

专为长篇网文创作设计的AI辅助技能。只需填入世界观设定和角色档案，就能逐章生成去AI痕迹的高质量网文。

## 核心特点

- **模块化开关**：19个功能模块（2必填+17可选），填什么启用什么，不填自动跳过
- **断点续写**：3个系统文件自动维护进度、摘要、仪表盘，随时中断随时续写
- **去AI痕迹**：10项默认检查+3类专属检查（古风/都市/修仙），支持自定义
- **上下文管理**：每章自动生成摘要，避免长篇连载时上下文爆炸
- **完整创作闭环**：从大纲到伏笔到爽点节奏，全链路管理
- **首次使用引导**：AI自动检测配置状态，引导用户逐步填写
- **一套技能适配所有网文类型**：穿越/修仙/都市/种田/系统流……换个设定就行

## 模块清单

### 🔄系统文件（3个，自动维护）

| 文件 | 说明 |
|------|------|
| 0. 当前进度 | 记录写作位置、角色状态、待办事项，AI启动先读这个 |
| 0. 章节摘要 | 每章自动生成摘要，后续章节基于摘要+前1-2章正文创作 |
| 0. 进度仪表盘 | 字数/爽点/伏笔/情感线一览，说「进度」即可查看 |

### ⚠️必填（2个）

| 模块 | 说明 |
|------|------|
| 1. 世界观设定 | 时代背景、地理、势力、规则体系 |
| 2. 角色档案 | 主角、配角信息，创作时自动更新状态 |

### ⚡可选（17个）

| 模块 | 说明 | 适用场景 |
|------|------|----------|
| 3. 伏笔管理 | 埋设/回收追踪，自动提醒未回收伏笔 | 有复杂剧情的 |
| 4. 章节大纲 | 卷→章→场景三级大纲树 | 需要整体规划的 |
| 5. 写作风格 | 视角、语言、节奏、描写侧重的定义 | 有固定风格的 |
| 6. 写作准则 | 禁忌词、一致性检查、自定义规则 | 有质量要求的 |
| 7. 去AI痕迹 | 自定义检查项，按类型专属配置 | 追求去AI效果的 |
| 8. 爽点系统 | 爽点类型库、密度控制、疲劳管理 | 追求"爽"感的 |
| 9. 金手指管理 | 能力详情、升级路径、使用限制、压制策略 | 主角有特殊能力的 |
| 10. 等级体系 | 等级序列、战力平衡、突破条件 | 升级流/战斗文 |
| 11. 章节钩子 | 5种钩子类型库，每章强制钩子检查 | 想留住读者的 |
| 12. 黄金三章 | 前三章开篇策略和检查清单 | 新书开局 |
| 13. 冲突密度 | 每章至少1冲突，升级曲线管理 | 避免平淡的 |
| 14. 字数管理 | 目标字数、上下限提醒、平台参考 | 需要控制字数的 |
| 15. 情感线 | 多情感线并行追踪，推进节奏管理 | 有CP线的 |
| 16. 时间线 | 全局时间轴，防时间矛盾 | 长篇连载 |
| 17. 水文检测 | 5项防水检查（重复/无效对话/灌水等） | 追求质量的 |
| 18. 世界观展开节奏 | 洋葱式揭示，防止设定轰炸 | 有复杂世界观的 |
| 19. 参考资料库 | 历史大事件/装备/技能/科技树/官职/丹方 | 有结构化数据的 |

## 触发方式

| 你说 | 发生什么 |
|------|----------|
| 「开始创作」「写第X章」「继续写」 | 进入完整创作流程（单章） |
| 「写第X到Y章」「一口气写N章」「批量写」 | 进入批量创作流程（多章连续） |
| 「预览第X章」 | 预览模式（只看写作要点，不生成正文） |
| 「修改第X章」 | 进入章节修改流程 |
| 「进度」「状态」「写到哪了」 | 查看进度仪表盘 |
| 「加个角色」「改大纲」「调设定」 | 进入中途修改设定协议 |
| 「设定」「角色」「伏笔」「大纲」等 | 进入对应管理模块 |

## 创作流程

```
首次使用引导 → 创作前检查 → 风格确认 → 内容生成 → 质量检查 → 去AI痕迹 → 正文保存 → 档案更新（摘要/进度/仪表盘）→ 输出
```

每章生成后自动：保存正文到 `正文/`、生成章节摘要、更新进度追踪、更新仪表盘。

## 快速开始

1. 打开 `data/1. 世界观设定.md`，填写你的小说世界
2. 打开 `data/2. 角色档案.md`，填写主角和重要配角
3. 根据需要填写其他可选模块（3-19号）
4. 说「开始创作」，即可生成第一章
5. 或者：直接把大纲发给AI，让AI帮你填充所有配置

## 断点续写

每次AI启动时自动读取 `0. 当前进度.md` 和 `0. 章节摘要.md`，无需手动告知写到哪了。说「继续写」即可从中断处继续。
