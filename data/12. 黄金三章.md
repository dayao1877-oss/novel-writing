# 黄金三章

## 第一章：制造冲突 + 展示金手指
目标：500字内抓住读者

- [ ] 开篇即冲突（不要铺垫，直接进入矛盾）
- [ ] 主角身份/处境快速建立
- [ ] 金手指出现（结尾处最佳）
- [ ] 必须有钩子结尾

## 第二章：验证金手指 + 获得初步收益
目标：让读者"爽"到，确认这本书值得追

- [ ] 主角利用金手指解决一个实际问题
- [ ] 展示金手指的核心价值
- [ ] 初步收益（物质/地位/能力提升）
- [ ] 埋下主线悬念

## 第三章：引入主线悬念
目标：建立长期阅读期待

- [ ] 揭示更大的世界/矛盾
- [ ] 主角确立初步目标
- [ ] 第一个重要配角出场
- [ ] 卷末钩子（让读者想知道后面的故事）

## 三章总检查
- [ ] 三章内主角是否至少"爽"了一次？
- [ ] 读者是否清楚主角的目标？
- [ ] 是否有至少一个让读者印象深刻的名场面？
- [ ] 三章总计字数是否在6000-12000字之间？
