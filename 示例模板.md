# 小说设定示例模板

> 使用方法：选择一个最接近你小说类型的示例，复制后修改具体内容，发给AI说「帮我根据以下设定填写所有配置文件」，AI会自动填充 data/ 目录下的所有相关文件。
>
> **新功能**：AI 还会自动配置去AI痕迹检查项、初始化进度系统，填充后可直接开始创作。

---

## 示例一：穿越历史文

```
【小说类型】穿越历史文
【书名】大明逆臣
【核心设定】现代历史教授穿越到明末崇祯十年，成为一个穷书生，利用历史知识预知大势，步步为营改变命运

【时代背景】明朝崇祯十年（1637年），内忧外患，李自成起义，后金虎视眈眈
【地理环境】主角起点在河南一个小县城，后辗转开封、南京、北京
【势力体系】朝廷（东林党vs阉党余孽）、李自成起义军、后金/清军、地方豪绅
【关键物品】主角随身携带的现代钢笔（被古人视为神物）、一本现代印制的明史书

【主角】陈渊，28岁，现代大学历史系副教授，穿越后身份是河南陈家沟落魄书生
【性格】沉稳内敛，善于隐忍，表面温润实则城府极深
【金手指】现代历史知识（知道未来3年所有大事件走向）
【金手指限制】只知道历史大势，不知道细节；历史因他的介入开始偏移，越干预越不准

【核心配角】
- 苏婉清：县令之女，聪明果敢，与主角联姻后成为最强后盾
- 赵铁柱：江湖镖师，忠义豪爽，主角的武力保障
- 周延儒：当朝首辅，老谋深算，前期盟友后期对手
- 李岩：李自成军中谋士，与主角惺惺相惜

【主线】主角从穷书生起步，先经商积累财富，再入仕途，试图在明末乱世中保全家族、改变历史走向
【爽点类型】知识碾压（预知天灾人祸布局）、技术降维（引入现代管理/工艺）、权谋反杀
【情感线】主角与苏婉清从政治联姻到真心相爱，慢热型
【写作风格】第三人称有限视角，语言偏古风但不晦涩，节奏张弛有度
【参考作者】月关、孑与2

【写作准则】
- 禁止现代网络用语混入古代对话
- 官职称呼必须符合明朝制度
- 历史大事件时间线必须与史实一致（偏离处需说明原因）

【去AI痕迹配置】
- 启用古风文专项检查（现代词汇/现代对话/称呼不当）
- 重点关注结尾说教式总结和情绪模板化

【历史大事件参考】
- 1637年：清军第四次入塞
- 1638年：清军攻破济南
- 1640年：河南大旱，人相食
- 1641年：李自成攻占洛阳，杀福王
- 1642年：松锦大战明军惨败
- 1644年：李自成攻入北京，崇祯自缢

【每章字数】3000字
【目标平台】起点
```

## 示例二：修仙升级流

```
【小说类型】修仙升级流
【书名】万界仙途
【核心设定】废柴少年偶得上古传承，从底层修士一路逆天崛起，最终飞升成仙

【时代背景】架空修仙世界，万宗林立
【地理环境】东荒（起点）→中州（宗门）→北冥（秘境）→仙界（终局）
【势力体系】
- 正道：天剑宗（主角宗门）、丹鼎宗、符箓门
- 邪道：血魔宗、幽冥殿
- 中立：商盟、炼器师公会
【规则体系】修炼等级：炼气→筑基→金丹→元婴→化神→渡劫→大乘→飞升
【关键物品】上古玉佩（金手指载体）、天衍剑（后期主武器）

【主角】林辰，16岁，东荒小镇孤儿，天生灵根残缺被嘲笑为废柴
【性格】坚韧不服输，外冷内热，对敌人狠辣对朋友义气
【金手指】上古玉佩内藏完整传承空间，可加速修炼、炼丹、领悟功法
【金手指限制】每次使用消耗灵石，等级越高消耗越大；传承需要逐层解锁，每突破一个大境界才能开一层

【核心配角】
- 苏灵儿：天剑宗天才弟子，冰山美人，与主角从对手到伴侣
- 胖子王大锤：主角发小，炼器天才，搞笑担当
- 玄长老：天剑宗隐藏高手，主角引路人
- 血魔子：血魔宗少主，宿命对手

【主线】从废柴逆袭到飞升，每个大境界对应一个主要对手和一个秘境副本
【爽点类型】升级突破、打脸反杀、奇遇得宝、装逼炫技
【情感线】主角与苏灵儿从竞争到并肩作战到确认关系，中速推进
【写作风格】第三人称全知，语言简洁有力，快节奏
【参考作者】辰东、忘语

【写作准则】
- 每次突破必须有实质付出，不能凭空升级
- 禁止"恰好路过就捡到宝"式的巧合
- 反派必须有动机，不能为坏而坏

【去AI痕迹配置】
- 启用修仙文专项检查（修炼描写不千篇一律/战斗有画面感/功法不似说明书）
- 重点关注句式雷同和形容词堆砌

【等级体系】
| 等级 | 名称 | 寿命 | 战力描述 | 突破条件 |
|------|------|------|----------|----------|
| 1 | 炼气 | 120年 | 可御器飞行 | 感应灵气 |
| 2 | 筑基 | 200年 | 灵力凝实 | 筑基丹+悟道 |
| 3 | 金丹 | 500年 | 可御空千里 | 凝聚金丹 |
| 4 | 元婴 | 1000年 | 元神出窍 | 元婴化形 |
| 5 | 化神 | 2000年 | 可碎山裂地 | 神魂合一 |
| 6 | 渡劫 | 5000年 | 渡天劫 | 引动天劫 |
| 7 | 大乘 | 10000年 | 近仙之力 | 大乘圆满 |
| 8 | 飞升 | 不朽 | 超脱凡界 | 飞升仙界 |

【每章字数】3000字
【目标平台】起点
```

## 示例三：都市系统流

```
【小说类型】都市系统流
【书名】神豪从签到开始
【核心设定】普通上班族获得签到系统，每天签到获得金钱/技能/物品，从底层逆袭成为商业帝国掌控者

【时代背景】现代都市，2024年
【地理环境】主角起点在三线城市，后发展到一线城市
【势力体系】
- 商业：各大财团、上市公司
- 官方：政府关系网络
- 地下：灰色势力
【规则体系】签到系统规则：每日签到→随机奖励，连续签到有额外奖励，完成系统任务解锁新功能
【关键物品】系统面板（手机APP形式）

【主角】江辰，25岁，小公司职员，月薪5000，被女友劈腿后激活系统
【性格】低调务实，表面随和实则精明，不主动惹事但绝不吃亏
【金手指】签到系统
【金手指限制】每天只能签到一次，奖励随机不可控；系统任务失败有惩罚；使用系统能力时不能被他人发现异常

【核心配角】
- 林诗语：公司高管，冰山女总裁，后期伴侣
- 张大彪：退伍军人，主角保镖兼兄弟
- 陈少：富二代，前期反派，后期被打脸专业户
- 老王：系统提示的神秘NPC，后期揭示身份

【主线】从月薪5000到商业帝国，每个阶段对应一个商业领域和对手
【爽点类型】装逼炫技、打脸反杀、逆袭翻身、知识碾压
【情感线】主角与林诗语从上下级到合作伙伴到恋人，慢热
【写作风格】第一人称，轻松幽默，节奏快
【参考作者】我本纯洁

【写作准则】
- 对话必须有烟火气，不能太书面
- 商业逻辑要基本合理，不能"随便捡到100万"
- 装逼要适度，不能过度浮夸

【去AI痕迹配置】
- 启用都市文专项检查（对话书面化/网络用语过时/职场理想化）
- 重点关注缺乏口语化表达和对话过于礼貌完整

【系统功能解锁路径】
- Lv1：每日签到（金钱奖励）
- Lv5：解锁商城（可用签到积分兑换技能/物品）
- Lv10：解锁投资系统（预知涨跌）
- Lv20：解锁人脉系统（自动结识关键人物）
- Lv50：解锁终极功能（未知的最终能力）

【每章字数】2500字
【目标平台】番茄/七猫
```

## 示例四：种田穿越文

```
【小说类型】种田穿越文
【书名】农门女首富
【核心设定】现代农学博士穿越成古代农家女，凭借现代农业知识带领家族脱贫致富

【时代背景】架空古代，类似宋朝，商业相对发达
【地理环境】江南水乡小镇，后发展到府城、京城
【势力体系】
- 家族：林家（主角家族，贫农）、赵家（镇上富户）、王家（县城大族）
- 官方：县衙、府衙
- 商业：各路商人、行会
【规则体系】无战力体系，纯靠智慧和知识
【关键物品】主角随身空间（可储存物品、加速植物生长1:10）

【主角】林晚棠，24岁，现代农学博士，穿越成16岁农家女
【性格】聪慧果断，勤劳踏实，重亲情，有恩必报有仇必报
【金手指】随身空间（储物+时间加速种植）
【金手指限制】空间面积有限（初始1亩，随使用扩大）；时间加速消耗主角体力；不能放入活物

【核心配角】
- 奶奶：林家主心骨，慈祥但精明
- 林大哥：老实本分，种地好手
- 顾景行：落魄书生，后中举，主角夫君
- 钱掌柜：镇上布庄掌柜，主角第一个商业伙伴

【主线】从赤贫农家起步→种地改善生活→经商积累财富→涉足朝堂→成为女首富
【爽点类型】技术降维（现代农业知识碾压）、知识碾压、逆袭翻身
【情感线】主角与顾景行从互相帮助到日久生情，慢热温馨
【写作风格】第三人称有限，轻松温馨，节奏偏慢但不断有新东西
【参考作者】郁雨竹

【写作准则】
- 种田细节要真实，不能"种什么都大丰收"不解释
- 人际关系要符合古代宗族社会逻辑
- 商业发展要有阶梯式过程，不能一步登天

【去AI痕迹配置】
- 启用古风文专项检查
- 重点关注感官细节缺失（需要丰富农事/食物/布料等触感描写）

【科技树/工艺流程】
| 技术 | 前置条件 | 效果 | 引入章节 |
|------|----------|------|----------|
| 堆肥法 | 粪便+秸秆 | 产量+30% | 第3章 |
| 温室大棚 | 竹子+油纸 | 反季节蔬菜 | 第8章 |
| 肥皂 | 油脂+草木灰 | 日用品赚钱 | 第12章 |
| 嫁接技术 | 果树苗 | 品种改良 | 第20章 |
| 蒸馏酒 | 发酵+蒸馏 | 高端酒品 | 第30章 |

【每章字数】2500字
【目标平台】番茄/七猫
```

## 使用说明

1. 选择最接近你小说类型的示例
2. 复制该示例的全部内容
3. 把具体设定改成你自己的
4. 发给AI，说：**「帮我根据以下设定填写所有配置文件」**
5. AI会自动填充 `data/` 目录下所有相关的配置文件，并初始化进度系统

**AI会填写哪些文件？**
- ⚠️必填：`1. 世界观设定.md`、`2. 角色档案.md`
- ⚡按需：`3-19号` 中所有有信息填充的模块
- 🔄系统：自动初始化 `0. 当前进度.md`、`0. 进度仪表盘.md`

**如果你是大纲/章纲格式**，直接发给AI也可以，AI会自动解析并反问缺失信息。

如果您的小说是混合类型（如穿越+系统+种田），可以混合多个示例的字段，AI会自动识别并填写。
